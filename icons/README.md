# Extension Icons

## Current Icons (Variant 5 - Orange "Việt")
- `icon16.png` - 16x16 toolbar icon
- `icon48.png` - 48x48 popup/menu icon  
- `icon128.png` - 128x128 store icon

## Backup Icons (Variant 6 - Navy "En/Vi" chat bubbles)
To switch to backup icons, rename the backup files:
- `icon16_backup.png` → `icon16.png`
- `icon48_backup.png` → `icon48.png`
- `icon128_backup.png` → `icon128.png`

Then reload the extension in chrome://extensions
